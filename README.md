Tools for optimizing warera builds:
To test damage maximizer:
- Ensure you have python installed
- Download ```WE-optimizer``` (either via ```git pull https://github.com/Dtzibell/WE-optimizer``` or via the ```</>Code``` panel
- Navigate via the command line to where you downloaded the folder via ```cd```
- If you do not yet have uv: ```pip install uv``` via the command line
- execute ```uv sync``` via the command line
- execute ```cd damage_maximizer```
- execute ```uv run ui.py```
